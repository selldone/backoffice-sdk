# Commerce Cross-Modules

This folder documents concepts that appear across multiple modules: orders, incentives, marketplace, POS, and dropshipping.

## Shared Concepts

| Concept | Used by |
|---|---|
| Basket/order type | Process Center, Product detail, Customer orders, POS. |
| Product/category targeting | Incentives, Marketing, Products, Listing. |
| Vendor assignment | Marketplace, Products, Product detail. |
| Payment/gateway | Finance, Orders, POS, Affiliate POS. |
| Customer wallet | Finance, CRM, Orders. |

For details, read the corresponding module under `dashboard/shop`.
